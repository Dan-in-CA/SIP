SIP System updater plugin

This plugin compares the local SIP version with the GitHub repository and can update the local software.

It fetches version information from the GitHub SIP repository when the plugin is opened. This may take several seconds to complete depending on the speed of your internet connection.

You should then be presented with information about whether a new version of the software is available and what changes have been made.

To update SIP

  • Click the "UPDATE SIP" button at the lower left of the screen
  • If the Home screen does not appear after several seconds, click your browser's refresh button.

When you have been returned to the home screen, you should see the latest version number and date in the page footer.
